# Python-Password
## Use
* Download Zip or Clone Repo
* Run main.py:
  * On Macos And Linux Open Terminal And Navigate To The Folder Where The Project Is Then Run:
    ```
    python3 main.py
    ```
  * On Windows Open PowerShell And Navigate To The Folder Where The Project Is Then Run:
    ```powershell
    python main.py
    ```
